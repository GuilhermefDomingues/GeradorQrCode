# QR Code Generator

Projeto desenvolvido com HTML, CSS e JavaScript que permite gerar códigos QR a partir de textos ou URLs fornecidos pelo usuário. O sistema utiliza a biblioteca QRCode.js para criar os códigos dinamicamente e disponibiliza a opção de download.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- QRCode.js

## ✨ Funcionalidades

- Gerar QR Code em tempo real
- Suporte para URLs
- Suporte para textos personalizados
- Visualização instantânea
- Download do QR Code
- Interface moderna e responsiva
- Fácil utilização

## 📂 Estrutura do Projeto

```bash
QR-Code-Generator/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## 🎯 Como Executar

1. Clone ou baixe o projeto.
2. Abra a pasta do projeto.
3. Execute o arquivo `index.html`.
4. Digite uma URL ou texto.
5. Clique em "Gerar Código QR".
6. Faça o download do QR Code gerado.
